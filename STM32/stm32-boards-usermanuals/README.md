## Boards used
* [NUCLEO-L476RG](https://os.mbed.com/platforms/ST-Nucleo-L476RG/)
    * Used to deploy the Power System Stabilizer (PSS) Controller eFMU.
    * Referred to as the "Controller" board.
* [NUCLEO-G474RE](https://os.mbed.com/platforms/ST-Nucleo-G474RE/)
    * Used to deploy the eFMU for Real-Time simulation of the controlled power generator unit (Excitation System, Synchronous Generator) and the power grid.
    * Referred to as the "Plant" board.

**Note:** A copy of the *.pdf manuals for the boards used are provided in two sub-directories with the corresponding name of the board.
